# 📝 Task Management Dashboard

A modern, user-friendly task management application designed to help users organize their daily activities effectively with a clean and intuitive interface.

## 🌟 Key Features
* **Task Creation:** Easily add task titles and detailed descriptions.
* **Image Upload:** Supports a "Drop a Photo" feature to attach visual references to tasks.
* **Priority Levels:** Built-in priority categorization (**Low**, **Medium**, **Hard**).
* **Workflow Tracking:** Real-time organization of tasks into categories (**Hold**, **In Progress**, **Done**) with automatic counter indicators.
* **Live Search:** Integrated search bar for quick access to specific tasks.
* **Data Persistence:** Uses **Local Storage** to save tasks, ensuring your data is kept even after refreshing the page.
* **Responsive Design:** Fully responsive layout that adapts seamlessly to all screen sizes.

## 🛠 Tech Stack
* HTML5, CSS3, JavaScript
